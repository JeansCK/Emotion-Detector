# EmotionDetection

A lightweight Python package and API service for detecting emotions in text using IBM Watson NLP.

## Overview

EmotionDetection analyzes natural language input and identifies the dominant emotional tone.
It leverages the Watson NLP Emotion Predict model to classify text into five categories:

- joy
- anger
- disgust
- sadness
- fear

## The project includes:

- a reusable Python package
- a simple Flask-based API
- unit tests
- static code analysis compliance (PyLint 10/10)
